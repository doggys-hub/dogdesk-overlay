DoGDeskOS Dokumentaatio

Tervetuloa DoGDeskOS:n viralliseen dokumentaatioon. Täältä löydät tietoa paketeista, komennoista ja järjestelmän osista.

Poke Package Manager

poke-mini on DoGDeskOS:n kevyt paketinhallinta.

poke -i paketti

Asentaa paketin.

poke -r paketti

Poistaa paketin.

Heinola Runtime

Heinola Runtime (hrun) käynnistää Heinola-sovelluksia. 

heinola run hedit95

Käynnistää Heinola Edit 95:n.

Heinola Edit 95

Ensimmäinen Heinola Edit -tekstieditori. 

heinola run hedit95 tiedosto.txt

Avaa tai luo tekstitiedoston muokattavaksi.

Kanto Libraries

kanto-libs tarjoaa yhteensopivuuskerroksen muiden repositorioiden käyttöön.
