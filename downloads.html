Lataukset

Tältä sivulta löydät DoGDeskOS-repositorion asennusohjeet ja linkit.

Dogdeskos Overlay

Lisää repositorio järjestelmääsi:

eselect repository add dogdeskos git https://github.com/doggys-hub/dogdesk-overlay.git
emaint sync -r dogdeskos

Poke Package Manager

Asenna poke-mini:

emerge poke-repo/poke-mini

Tai myöhemmin Poken kautta:

poke -i poke-mini
